================
ackward::logging
================

``ackward.logging`` is a C++ interface atop Python's standard ``logging``
module. In general, you use ``ackward.logging`` just as you would use
the Python module. The function and class names have been kept as
close as possible to Python's to avoid forcing users to remember two
APIs.
